# Introduction for Terraform users

If you already know Terraform, you will recognize that NixOps resources are very similar to Terraform resources.

The main difference is that NixOps resources are defined in Nix, a functional programming language, whereas Terraform resources are defined in HashiCorp Configuration Language (HCL).
This language is more capable, and it is designed for the Nix package manager / build system, capable of subsuming all preceding stages of a legacy deployment pipeline. <!-- 🔥🚀 -->
