# Introduction for NixOps 1 / 2 users

If you already know NixOps 1 (or "2"), you will see that NixOps 4 is also capable of provisioning machines with NixOS and resources, but the design is quite different.

NixOps4 is designed as a platform for defining generic "resources", and it relies on resources providers and modules to define specifics like NixOS machines.

Note that the implementation is still in progress, so we have to cut the comparison here for now.
